# Base para projetos Backend

## Utilizando Node.js e Sequilize mySQL

## Project setup

```
npm install
```

Para liberar o acesso de outros dominios
npm i cors

### Compiles and hot-reloads for development

```

npm run backend

```
### JWT Autentication

```

npm i jsonwebtoken dotenv-safe

```
### Para encriptografar a senha

```
npm i bcrypt

```
